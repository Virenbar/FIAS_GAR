# Таблицы ГАР

Описание таблиц и их полей

## Описание общих полей

### `OBJECTID`

Глобальный уникальный идентификатор объекта (`LONG`)  
Внутренний идентификатор, используется для связи между таблицами.

### `OBJECTGUID`

Глобальный уникальный идентификатор объекта (`GUID`)  
Внешний идентификатор, может использоваться в других системах.

### `ISACTUAL`

Принимает значение:

* 0 – не актуальный
* 1 – актуальный

У последней записи адресного объекта элемент всегда принимает значение 1, у предыдущих – значение 0

### `ISACTIVE`

Принимает значение:

* 0 – недействующий адресный объект  
* 1 – действующий
